# GameTracker

Static Bootstrap checklist tracker for the Ghost in Machines Unreal Engine development plan.

## Contents

- `index.html` - tracker page
- `styles.css` - Material Dark Theme styling
- `app.js` - progress tracking, password-gated editing, image support, import/export
- `checklist-data.js` - extracted checklist data from the planning PDF
- `Documents/` - source planning PDFs

Open `index.html` in a browser to run the tracker.
