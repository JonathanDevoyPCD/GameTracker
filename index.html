Production Direction

Roadmap Overview

Foundation First

Lock the concept, repo, Unreal project, reusable data, and core architecture before building feature-heavy systems.

Prototype Loop

Build movement, interaction, inventory, survival pressure, world blockout, and first threat encounters in small test maps.

Vertical Slice

Connect the Safe Zone, Maze, NPCs, machine encounter, event sequence, UI, audio, and death/outcome flow into one playable run.

Steam Path

Only after the slice works, move toward alpha expansion, beta feedback, performance, Steam assets, builds, and launch support.
